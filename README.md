## Lab 1
In Lab 1, I learned how to implement basic logic gates on a breadboard by using specific Integrated Circuits (ICs). For example, the 7408 IC was used for the AND gate, the 7432 IC for the OR gate, and the 7404 IC for the NOT gate. By correctly connecting these ICs with GND and VCC, switch(input) and Led(output).I was able to observe the behavior of each logic gate in real hardware. Although only three basic gates were used, I successfully combined them to construct a more complex combinational logic circuit. This lab enhanced my understanding of how theoretical logic gate concepts are translated into practical hardware implementations and improved my hands-on skills in circuit wiring and troubleshooting.

## Lab 2
In Lab 2, I learned how to download and use the Deeds Simulator to design and simulate digital logic circuits. Deeds allows logic circuits and timing diagrams to be created and tested without manually drawing them on paper. By simply adjusting the input bits during simulation, the output can be observed automatically in real time. This tool greatly improves efficiency and reduces human error when analyzing circuit behavior.
